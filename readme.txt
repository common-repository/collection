== Description ==

Collection is a plugin allowing and simplifying the creation of custom post type and metabox.
NOTE (beta): This plugin is very felt, it is still in development phase!

= Features Include =
* Create collections of post.
* Adding category and tags dedicated to collections.
* Add metabox dedicated to collections.
* Add information to content with shortcode.
* Take control of Divi projects.

== Configuration ==

1. Activate the plugin through the "Plugins" menu in WordPress.
2. Go to the plugin setting page, add your first collection and validate the changes. From your new collection exists and you can find it in the left admin menu of wordpress. You can also add a link in the different menus of the site.
3. Click on the metabox tab. You will find your different collections and you will be able to assign them metabox personalized to each one of them.
4. Now everything is ok, you can add items to your collections.
